# 🏛️ Enochian Governor Generator

A Bitcoin L1-native system for generating and managing 91 Governor Angels with authentic mystical knowledge from 18 traditions.

## Core Architecture

```
🏛️ THE LIGHTHOUSE (Knowledge Base)
    ↓ feeds wisdom to ↓
👑 GOVERNOR ANGELS (91 Unique AI Entities) 
    ↓ generate ↓
🎮 GAME CONTENT (Storylines/Events/Challenges/Riddles)
    ↓ delivered through ↓  
🌐 INTERACTIVE EXPERIENCES (Web/Game Interfaces)
```

## Project Structure

- `/core` - Core system components
  - `/governors` - Governor profiles and trait generation
  - `/lighthouse` - Knowledge base and retrieval systems
  - `/mystical_systems` - Implementation of mystical traditions
  - `/questlines` - Dynamic quest generation
  - `/utils` - Bitcoin integration and utilities
  - `/game_assets` - Visual and gameplay elements

- `/data` - Source data and knowledge seeds
- `/docs` - Architecture and implementation docs
- `/tests` - Test suites for core components
- `/engines` - Procedural generation engines
- `/onchain` - Bitcoin L1 integration

## Technical Specifications

- Zero web dependencies
- Bitcoin-native randomness
- Ordinals-compatible (400kb limit)
- WASM-compiled core
- Deterministic generation
- O(1) verification complexity

## Setup

1. Install Rust and wasm-pack
2. Run `wasm-pack build`
3. Run tests: `cargo test`
